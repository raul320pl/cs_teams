# cs_teams
Code for evaluate player skills and form teams for CounterStrike players.

1. Put data into data.json file:

```
{
 "config":{
	"show_teams" : 2,
	"comm": 0.5,	
	"maps":0.5,
	"team_play": 6.0,
	"skill": 1.0	
},
 "players":[	
{
	"name" : "Yego",
	"skills": {
		"comm" : 9,
		"skill" : 9,
		"maps" : 8,		
		"team_play" :10
		}
}		
]
}
```        

2. Run code and chose team proposals
